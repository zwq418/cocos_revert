# cocos_revert
一秒还原cocos creator打包后的游戏素材

# 需要安装node环境
https://nodejs.org/en

# 使用方法
在input目录放入你的文件，一般把整个assets文件放在input目录下，
## 还原spine素材
执行命令 node reversespine.js，
## 还原resources文件夹的图片、声音等普通素材
执行命令 node reversebundle.js，

输出整理好的素材到output目录
